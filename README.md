# 🚀 What I'm Building

### Zyqual

Zyqual is a national school improvement platform that supports:

• Lesson observations  
• Teacher appraisals  
• School performance analytics  
• Curriculum intelligence  
• Regional and national education dashboards

The platform enables data-driven improvement across the full education hierarchy:

Teacher → Principal → Education Officer → Regional Director → Ministry of Education
