# 👋 Hi, I'm Leon Pennicooke

Founder of **Zyqual**, an education intelligence platform designed to improve teaching quality, school leadership, and national education systems through data and technology.

I combine **education leadership and software engineering** to build platforms that help governments, schools, and teachers make better decisions using real data.

---

# 🚀 What I'm Building

### Zyqual

Zyqual is a national school improvement platform supporting:

- Lesson observations
- Teacher appraisals
- School performance analytics
- Curriculum intelligence
- Regional and national education dashboards

The system connects the full education structure:

Teacher → Principal → Education Officer → Regional Director → Ministry
